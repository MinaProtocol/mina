Adds an optional --signature-kind CLI argument to commands:

- `mina internal run-prover`
- `mina internal run-snark-worker-single`
- `mina internal test-genesis-block-generation`
- `mina advanced constraint-system-digests`
- `mina advanced compute-receipt-chain-hash`
- `delegation_verify` app
- `ocaml-signer` app
- `rosetta` app

This uses a default provided by the build profile.
