Introduce `mina-bootstrap`, a standalone Go CLI that automates node-operator pre-staging steps (downloading and loading archive database dumps, fetching precomputed blocks). It is shipped both as a Debian package (`mina-bootstrap`, depending only on `postgresql-client` and `ca-certificates`) and as a self-contained Docker image (`Dockerfile-mina-bootstrap`), and is wired into the Buildkite artifact pipeline. Because the tool requires Go 1.21 (`log/slog` + the modern GCS SDK) while the Bullseye builder image ships Go 1.19, `scripts/ensure-go.sh` provisions a pinned Go 1.21 toolchain for the `.deb` build path; the Dockerfile pins `golang:1.21`.
